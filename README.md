## ChroNiyam

Minimal four-quadrant time management matrix centered on the page. Everything else has been removed.

### Stack
- React 19, TypeScript, Vite
- ESLint (flat config)

### Run it
- Install: `npm install`
- Dev: `npm run dev`
- Lint: `npm run lint`
- Build: `npm run build`

### Structure
- `src/App.tsx` – renders the four quadrants
- `src/index.css` – layout and quadrant styling

### Notes
- Quadrants cover the classic urgent/important matrix: Do Now, Plan, Delegate, Eliminate.
